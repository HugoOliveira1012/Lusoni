# Lusoni
LuSoni Confeitaria Artesanal
Este é o site oficial da Lusoni, uma confeitaria artesanal localizada em Pinhais/PR. O projeto foi desenvolvido com foco em apresentar o catálogo de bolos e salgados de forma elegante, proporcionando uma experiência de navegação fluida tanto em computadores quanto em dispositivos móveis.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização personalizada, uso de Flexbox, Grid e Media Queries para responsividade.

JavaScript: Lógica de scroll suave e interatividade.

🛠️ Funcionalidades
[x] Design Responsivo: Adaptado para celulares, tablets e desktops.

[x] Catálogo Visual: Exibição otimizada de fotos de bolos (formato vertical) e salgados.

[x] Integração com WhatsApp: Botão direto para encomendas com mensagem automática personalizada.

[x] Navegação Inteligente: Menu com âncoras para seções específicas da página.

O site está disponível online através do GitHub Pages.
